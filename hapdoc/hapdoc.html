@click.group(invoke_without_command=True)
@click.option(
    '-v/--version', 'show_version',
    default=False, type=bool
)
def hapdoc(show_version: bool):

HapDoc is a powerful command-line interface tool that automates the process of generating documentation for various types of projects. With HapDoc, you can easily create high-quality documentation for Python, FastAPI, Vue, and many other types of projects.

@hapdoc.command()
def tmpl_list():

This command displays a list of all templates that have been saved by the current user. It is useful for quickly checking which templates are available for use. 

@hapdoc.command()
def tmpl_new():

The tml_new command creates a new template and saves it for future use. The user will be prompted to provide a name for the template and specify whether Tailwind CSS should be included.
